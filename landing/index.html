痛点

读标准，不该这么累

IEC / ISO / EN / GB / UL / ASTM / IEEE……每份标准都是数百页的专业文本，反复查术语、跳条款、理逻辑，耗时又费神。

术语密密麻麻

专业缩写和生僻词汇反复打断阅读节奏，查完词典又忘了上下文。

条款跳来跳去

"see Clause 5.2.3" 出现一次，就要手动翻页、定位、再找回原来的位置。

要求难以落地

读完后仍要手动整理 "测什么、怎么测、判定依据"，效率低下。

隐私安全

你的文件，始终属于你

检测认证行业的标准文件往往涉及保密要求，我们把"本地优先"作为第一性原则。

🔒

PDF 不上传云端

文件仅在本地读取和渲染，不会发送给任何第三方服务器。

✂️

仅发送选中片段

只有用户主动选中的文本才会发送给你配置的 LLM API。

🗝️

API Key 进系统钥匙串

API Key 通过 Rust keyring 存入系统安全存储，settings.json 中不留明文。

💾

数据本地持久化

批注、解读记录、暂存片段按 PDF 文件 hash 保存到本地 AppData。

1

下载并安装
